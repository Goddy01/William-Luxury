# William Broker Yacht & Boat

Premium bilingual (IT/EN) marketing prototype for yacht charter, helicopter transfers, private NCC, and brokerage on the Amalfi Coast.

## Stack

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- next-intl (Italian default, English)
- Framer Motion

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — redirects to `/it`.

## Deploy

Deploy to Vercel. Set `NEXT_PUBLIC_SITE_URL` to your production URL when ready.
